# String matching problem

Data Mining course - HCMUT - Semester 2 - 2022-2023

## Installation

- OpenJDK 11
- Dependencies (macOS/Linux)

```sh
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

- Dependencies (Windows)

```sh
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```

## Run the project

```sh
python run.py
```
